El juego consta de un caballero el cual va recolectando mascaras para ganar puntos
Evita a los enemigos para llegar a la luz final y terminar el nivel
el movimiento es utilizando las flechas del teclado <- -> 
utiliza la barra espaciadora para saltar
presiona R si gustas reiniciar el nivel
presiona S para guardar el progreso del nivel
presiona L para cargar el progreso guardado
